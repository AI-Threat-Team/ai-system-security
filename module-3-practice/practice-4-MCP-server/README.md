# Пратика пентестинга mcp сервера

1. Установите уязвимый mcp сервер в соответствии с интрукциями https://github.com/harishsg993010/damn-vulnerable-MCP-server/blob/main/docs/setup.md

2. Запустите сценарий 
```
cd challenges/easy/challenge2
python server_sse.py
```

3. Запустите mcp inspector
```
npx @modelcontextprotocol/inspector
```
